Projects

Movie Night Planner

Language/Tools: C#, WinForms

Course: C# II - Final Project

A desktop application that helps users organize and manage movie nights. Users can create a list of movies, vote on their top choices, and finalize plans based on the winning selection. The app includes full administrative controls for managing movies, votes, and event details. Demonstrates skills in UI design, data handling, and user interaction within a Windows Forms environment.

Multiplayer Number Guessing Game

Language/Tools: JavaScript, HTML, CSS

Course: JavaScript Programming - Final Project

A turn-based browser game for up to four players, where each player takes turns guessing a randomly generated number. The game tracks each player's number of attempts and successful guess, then displays a ranked results screen at the end. Built entirely in JavaScript, it showcases game loop logic, input validation, DOM manipulation, and dynamic UI updates.

Yahtzee (Single-Player Game)

Language/Tools: Python

Course: Python Programming - Final Project

A terminal-based Yahtzee game for single-player use. Players roll and re-roll dice, selecting which to keep while aiming for combinations like Full House or Yahtzee. The game includes a scoreboard, input validation, and scoring logic, using Python loops, conditionals, functions, and basic data structures.

HR Trak Application

Language/Tools: ColdFusion, SQL, JavaScript, HTML, CSS

Work Project

A web application built to help HR staff manage and track the employee onboarding process. It includes search pages for viewing open, closed, and archived positions, an import page for assigning workloads, and edit pages for updating onboarding progress. Features include multi-step forms, conditional input logic, and responsive search tools. The app helps reduce manual errors, improve team coordination, and make the process more transparent.
